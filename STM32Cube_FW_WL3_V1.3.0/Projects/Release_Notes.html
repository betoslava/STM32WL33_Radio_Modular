Release Notes for  STM32CubeWL3 Firmware Projects 

Copyright © 2024-2025 STMicroelectronics

Purpose

The projects are a set of applicative tasks that aim to explain and provide use cases of the different products features based on the products hardware (Boards, STM32 Core(s) features, interconnections and peripherals) and built around the different firmware components. They can be directly used or adapted by the STM32CubeWL3 Firmware Package users. They provide basic functionalities, show main features or demonstrate performance. They are provided with preconfigured projects for the main supported toolchains.

In the STM32CubeWL3 Firmware Package, the projects are organized in a sequential scheme based on the following levels:

  • Boards : Hardware platform (nucleo boards)
  • Project Category:
    • Applications
    • Demonstrations
    • Examples
    • Examples_LL
    • Examples_MIX
    • Templates
    • Templates_LL

The exhaustive list of projects is provided in this table: STM32CubeProjectsList.

Update History

Main Changes

Release of STM32CubeWL3 projects supporting STM32WL3x devices

  • Added folders and projects for NUCLEO-WL3RKB1 and NUCLEO-WL3RKB2 boards (check STM32CubeProjectsList for details)

  • MRSUBG_Timer demonstration

    • Optimized deep-stop exit timing for MRSUBG_TIMER interrupt
    • Add recovery mechanism for missed IRQ
    • Add wait for accurate MRSUBG_TIMER value

  • Added a PWR_ULTRA_DEEPSTOP example (NUCLEO-WL3RKB1/KB2)

    • Illustrates how to exercise WL3R UltraDeepStop low power mode

  • Added a PWR_SHUTDOWN example (NUCLEO-WL3RKB1/KB2)

    • Illustrates how to enter WL3R ShutDown low power mode and to detect the wakeup pin on exit

Supported Devices and boards

  • STM32WL3x devices
  • NUCLEO-WL33CCx boards
  • NUCLEO-WL3RKBx boards


Known Limitations

  • Sigfox applications are not available for the ARM KEIL toolchain
  • Examples for the following modules are only provided for the NUCLEO-WL3CC board
    • COMP, CORTEX, DMA, FLASH, I2C, IWDG, RCC, RTC, RNG, SPI, UART, USART


Dependencies

  • STM32CubeMX V6.16.0
    • Projects (Applications and Examples) are generated using STM32CubeMX version V6.16.0.

Note: When generating with CubeMX for the STM32CubeIDE toolchain, the project startup file is named differently compared to the one present in the CubeFW SDK. However, contents are the same.

Backward compatibility

  • MRSUBG structure names have been changed, adding a “_t” suffix for better readability. Related examples have been adapted accordingly.
  • MRSUBG: The IS_196MHZ definition has been replaced with STM32WL33XA. A legacy alias has been added to ensure compatibility with existing projects.
